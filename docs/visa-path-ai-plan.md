## Core Features (9-Day MVP)

### 1. **AI Intake & Profile Builder** _(Day 1-2)_

- User inputs: current country, target country, profession, experience, education
- Claude AI analyzes eligibility against 50+ visa types (work, study, entrepreneur, digital nomad, etc.)
- Returns: Top 3 recommended visa pathways ranked by success probability, cost, and processing time

### 2. **Smart Document Assistant** _(Day 2-4)_

- AI scans user documents (passport, degree, work history) via vision model
- Auto-detects missing documents specific to each visa type
- Generates AI-drafted cover letters, motivation statements, CVs tailored to visa requirements
- Flags compliance risks (salary thresholds, language requirements, background checks)

### 3. **Visa Timeline Forecaster** _(Day 4-5)_

- Fine-tuned model on real visa processing data (EU Blue Cards, US H1-B, Canada Express Entry, UK Skilled Worker, etc.)
- Predicts processing time based on country pair, visa type, season, current queue length
- Shows: estimated approval date, alternative faster routes, risk factors
- Real-time updates as new data arrives

### 4. **Cost Calculator & Budget Planner** _(Day 5-6)_

- Calculates total relocation cost: visa fees + legal fees + moving + housing deposit + COLA
- Compares country pairs (e.g., "Germany vs. Canada for Software Engineer")
- Shows ROI: salary difference vs. relocation cost
- Suggests financial optimization (e.g., "Study visa ‚Üí work permit is 30% cheaper than direct work visa")

### 5. **Interactive Visa Pathway Map** _(Day 6-7)_

- Visual flowchart showing:
  - Current situation ‚Üí Target role ‚Üí Visa types ‚Üí Approval timeline ‚Üí Country benefits
  - Alternative routes with trade-offs
  - Risk hotspots & mitigation strategies
- Dark mode support, mobile-responsive
- Shareable pathway links for advisors/employers

### 6. **AI Chat Agent** _(Day 7-8)_

- Claude-powered conversational interface
- Answers: "What's the fastest visa to work in Germany?", "Can I extend my current visa?", "What countries need my skills?"
- Learns from user context across the session
- Generates weekly digest emails for tracking

### 7. **Community & Advisor Marketplace** _(Optional, Day 8-9)_

- Users can share successful pathways
- Immigration advisors can verify accuracy (add trust score)
- Monetization hook: paid advisor referrals
